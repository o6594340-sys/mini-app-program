💾 Экспорт и импорт данных

Сохраните все настройки и контент в файл — или восстановите из ранее скачанного файла.

⬇️
Скачать бэкап
Сохраняет всё: программу, отель, кухню, историю и брендинг. Файл .json — откройте позже для восстановления.
⬆️
Загрузить бэкап
Восстановит данные из файла. Текущие настройки будут перезаписаны — убедитесь, что файл от этого проекта.
🗑️
Сбросить к данным по умолчанию
Удалит все сохранённые данные из браузера. Приложение вернётся к содержимому из data.js. Брендинг и оформление сохранятся.

🗂 Новый проект

Каждое мероприятие — отдельное приложение со своей ссылкой. Проекты существуют параллельно и не влияют друг на друга.

📄
Скачать шаблон
Скачает чистый data.js с пустыми полями. Вставьте в новый репозиторий — и заполните данными нового мероприятия.
🐙
Скопировать репозиторий
Откроет GitHub — нажмите «Use this template» → введите название нового проекта → включите Pages. Готово.
Открыть на GitHub →
📋 Как создать новый проект за 5 шагов
1
Скопируй репозиторий — кнопка «Открыть на GitHub» → «Use this template» → назови, например mini-app-dubai
2
Включи GitHub Pages — Settings → Pages → Source: main → Save
3
Скачай шаблон data.js — заполни данными нового мероприятия
4
Загрузи data.js в новый репозиторий (заменив старый)
5
Готово! Новый сайт: https://o6594340-sys.github.io/mini-app-dubai/
